บทที่ 6 — Economic Load Dispatch (ELD)

Cost Function & Incremental Cost (IC)

F_i(P_i) = a_i + b_i·P_i + c_i·P_i² [฿/h หรือ $/h] IC_i = dF_i/dP_i = b_i + 2·c_i·P_i [฿/MWh] เงื่อนไขเหมาะสม: λ* = IC_1 = IC_2 = ··· = IC_n P_i = (λ* − b_i) / (2·c_i)

Lambda Iteration Algorithm

  1. สมมติ λ เริ่มต้น
  2. คำนวณ P_i = (λ − b_i)/(2c_i) แต่ละ unit
  3. ตรวจ P_i,min ≤ P_i ≤ P_i,max (ถ้าเกิน → fix ที่ limit)
  4. ตรวจ ΣP_i = P_D (demand) ← ถ้าไม่เท่า → ปรับ λ
  5. ถ้า ΣP_i < P_D → เพิ่ม λ | ถ้า ΣP_i > P_D → ลด λ
⚠ Unit ที่ถูก fix ที่ limit → ไม่ต้องคิด IC แล้ว ลบออกจาก lambda loop

Priority-List Method (Field Cost Base List)

Average Cost = F_i(P_test) / P_test [฿/MWh] P_test = Min / Average / Max ← อาจารย์กำหนดมาให้เลือก 1
  1. คำนวณ Avg Cost แต่ละ unit ที่จุด P_test ที่กำหนด
  2. เรียงลำดับ: ต้นทุนต่ำสุด → เริ่มใช้ก่อน (Priority 1)
  3. เพิ่ม unit ทีละตัวจนครบ demand แต่ละ time period
ตัวอย่าง: F(P) = 500 + 7P + 0.01P² ที่ P_avg = 200 MW → F(200) = 500 + 1400 + 400 = 2300 ฿/h Avg Cost = 2300 / 200 = 11.5 ฿/MWh → เปรียบเทียบกับ unit อื่น

IC-Based Priority

IC = b + 2c·P_test [฿/MWh] IC ต่ำ → Priority สูง (เริ่มใช้ก่อน)

บทที่ 7 — AGC (Automatic Generation Control)

ACE — Area Control Error

ACE_i = ΔP_tie,i + B_i × Δf
ΔP_tie,i = กำลังส่งออกจริง − กำลังส่งออกตามกำหนด (Scheduled) B_i = Frequency Bias Setting [MW/Hz] Δf = f_actual − f_nominal [Hz] (ปกติ = 0 ที่ steady state)

2-Area System

ACE_1 = ΔP_tie,12 + B_1 · Δf ACE_2 = ΔP_tie,21 + B_2 · Δf โดย: ΔP_tie,21 = −ΔP_tie,12 (กระแสออกจาก Area1 = เข้า Area2)

3-Area System

ACE_i = Σ(ΔP_tie,ij สุทธิของ Area i) + B_i · Δf ΔP_tie,i = กำลังส่งออกสุทธิ Area i = Σ ΔP_ij (+ = ส่งออก, − = รับเข้า)

การแปลผล ACE → Action

ACEความหมายAction
ACE > 0ผลิตมากเกิน / ส่งออกเกินลด generation ↓
ACE < 0ผลิตน้อยเกิน / รับเข้าเกินเพิ่ม generation ↑
ACE = 0สมดุลคง generation

Power Balance ใน Area

ΔP_G,i − ΔP_D,i = ΔP_tie,net,i ΔP_tie,net = ΔP_G − ΔP_D ถ้า ΔP_D เพิ่ม (disturbance) → ΔP_tie,net เป็นลบ → ACE < 0 → เพิ่ม Gen

Steady-State Frequency (2-Area)

Δf_ss = −ΔP_D_total / (β_1 + β_2) β_i = D_i + 1/R_i [MW/Hz] (Freq Bias Total) ΔP_tie,12_ss = β_2·ΔP_D1 / (β_1+β_2) − ΔP_D1 (ถ้า ΔP_D เฉพาะใน Area1)

⚡ Ch8 — Touch/Step Voltage & Ground Grid Resistance

บทที่ 8 — Touch & Step Voltage (IEEE 80)

Allowable Touch Voltage

E_touch = (1000 + 1.5·Cs·ρs) × K / √t_s [V]

Allowable Step Voltage

E_step = (1000 + 6·Cs·ρs) × K / √t_s [V]
K = 0.116 (คน 50 kg) หรือ 0.157 (คน 70 kg) Cs = derating factor ของผิวชั้นบน (= 1.0 ถ้าไม่มี surface material) ρs = soil resistivity ผิวชั้นบน (Ω·m) — ถ้าไม่มี gravel → ใช้ ρs = ρ t_s = fault clearing time (s) — ยิ่งเร็ว ยิ่งทนได้มากขึ้น

การแปลผล

เงื่อนไขผล
E_measured ≤ E_allowable✓ SAFE — ปลอดภัย
E_measured > E_allowable✗ UNSAFE — ต้องแก้ไข
⚠ ข้อสอบมักให้ E_touch_measured และ E_step_measured มาเปรียบ เลือก K ตาม body weight ที่โจทย์กำหนด

บทที่ 8 — Ground Grid Resistance (R_G)

Sverak Formula (IEEE 80)

R_G = ρ × [ 1/L_T + 1/√(20·A) × (1 + 1/(1 + h·√(20/A))) ] [Ω]
ρ = ความต้านทานดิน (Soil Resistivity) [Ω·m] L_T = ความยาวรวมของสายกราวด์ทั้งหมด [m] A = พื้นที่กริด (กว้าง × ยาว) [m²] h = ความลึกของการฝัง (burial depth) [m]
⚠⚠ อาจารย์เน้น: ระวังการแทนค่า h ให้ถูกต้องในสมการ — อ่านโจทย์ให้ดีว่าให้ h มาเป็น m หรือ ft (แปลงหน่วยก่อน)

วิธีอ่านโจทย์กริด

  1. นับจำนวนตาราง → หาความกว้าง (W) และความยาว (L) ของกริดรวม → A = W × L
  2. นับเส้น Conductor → L_T = (จำนวนแนวนอน × L) + (จำนวนแนวตั้ง × W)
  3. อ่านค่า h (burial depth) และ ρ จากโจทย์
  4. แทนสมการ Sverak

ตัวอย่างเร็ว: กริด 3×3 (spacing 10m), h=0.5m, ρ=100 Ω·m

A = 20×20 = 400 m² L_T = 3 แนวนอน×20m + 3 แนวตั้ง×20m = 60+60 = 120 m (แต่ตัวอย่างนี้สำหรับฝึกดูสูตร — โจทย์จริงให้ค่ามาเลย)

SBO — Select Before Operate

Step 1 SELECT: เลือกอุปกรณ์ → ระบบตรวจ Interlock → "Selected OK" + เปิด Timer 10-30s Step 2 OPERATE: กดคำสั่ง (OPEN/CLOSE) → ระบบตรวจ Interlock อีกครั้ง → ส่งคำสั่ง → Feedback TIMEOUT: ถ้าเกินเวลา → ต้อง Select ใหม่

SBO ช่วยลด Human Error อย่างไร?

  • ป้องกันคลิกผิด Device — บังคับเห็นชื่ออุปกรณ์ชัดใน Select step
  • ระบบตรวจ Interlock Logic ก่อนส่งคำสั่งทุกครั้ง
  • Timeout ป้องกัน Double Command / คำสั่งค้างเก่า
  • BCU (Bay Control Unit) double-check ก่อน Operate
  • ถ้าไม่มี SBO → DS ปิดขณะมีไฟ → Arc Flash, Breaker Trip ผิด → Blackout

Substation Gateway — หน้าที่

  1. Protocol Conversion — IEC 61850 ↔ IEC-104/DNP3/Modbus
  2. Data Concentration — รวมข้อมูล IED → SCADA
  3. Event & Timestamp — SOE, Event Buffer
  4. Control Command — SCADA → Field Devices
  5. Cybersecurity — Authentication, Encryption, Firewall

⚡ Ch10 — Protection Relay Setting (วิธีเลือกค่าทั้งหมด)

บทที่ 10 — Relay Setting ขั้นตอน

Step 1: เลือก CT Ratio (CTR)

CTR = I_load_primary / I_secondary_rated ปกติ I_secondary_rated = 5 A CTR = I_load / 5 → ปัดขึ้นเป็น Standard Ratio Standard CT Ratio: 50/5, 75/5, 100/5, 150/5, 200/5, 300/5, 400/5, 500/5, 600/5, 800/5, 1000/5
ตัวอย่าง: I_load = 240 A CTR = 240/5 = 48 → เลือก 50/5 (CTR = 10) หรือ 100/5 (CTR = 20) ถ้าต้องการ margin มากกว่า

Step 2: เลือก Pick-up Current (I_p)

I_p = K × I_load_secondary = K × (I_load / CTR) K = 1.25 ถึง 1.5 (ต้องไม่ trip ขณะ normal load) I_p ต้อง < I_fault_minimum_secondary / 2 (เพื่อ sensitivity) I_load_secondary = I_load / CTR [A secondary]
ตัวอย่าง: I_load = 240A, CTR=50/5=10 I_load_secondary = 240/10 = 24 A → ผิดปกติ (relay rated 5A) หมายเหตุ: I_load_secondary = I_load/CTR ต้องไม่เกิน 5A ใน normal load
⚠ I_p ต้องเป็น tap setting ที่มีบน relay (เช่น 50%, 75%, 100%, 125%, 150%, 175%, 200% ของ 5A)

Step 3: คำนวณ PSM

PSM = I_fault_secondary / I_p_set = (I_fault / CTR) / I_p
I_fault = กระแสลัดวงจรที่จุดป้องกัน (จากโจทย์) I_fault_secondary = I_fault / CTR I_p_set = ค่า pick-up ที่ตั้งบน relay [A] PSM ต้อง > 1.0 เพื่อให้ relay ทำงาน

Step 4: เลือก TMS

TMS = 0.025 ถึง 1.0 (ปกติ) เลือกให้ relay trip ภายในเวลาที่ต้องการ → หรือ graded กับ downstream relay (CTI = 0.3–0.5 s)

Step 5: คำนวณ Trip Time (T)

⚠ ต้องรู้ว่าโจทย์ใช้ IDMT Curve แบบไหน — ถามอาจารย์หรือดูจาก relay spec

IEC/BS IDMT Relay Curves

Standard Inverse (SI): T = TMS × 0.14 / (PSM^0.02 − 1) Very Inverse (VI): T = TMS × 13.5 / (PSM − 1) Extremely Inverse (EI): T = TMS × 80 / (PSM² − 1) Long Time Inverse (LTI): T = TMS × 120 / (PSM − 1) Definite Time: T = TMS (fixed)

ตัวอย่างการคำนวณ

โจทย์: I_load=300A, I_fault=3000A, ต้องการ T=0.3s 1. เลือก CTR = 300/5 = 60 → ใช้ 300/5 standard (CTR=60) 2. I_load_sec = 300/60 = 5A → เลือก I_p = 125% × 5 = 6.25A (เลือก tap 125% = 1.25× 5A = 6.25 A) 3. I_fault_sec = 3000/60 = 50A PSM = 50 / 6.25 = 8.0 4. เลือก TMS = 0.1 (ลองก่อน) 5. Very Inverse: T = 0.1 × 13.5/(8−1) = 0.1×1.929 = 0.19s ถ้าต้องการ 0.3s → TMS = 0.3/1.929 = 0.156 → เลือก TMS=0.15 หรือ 0.2

Ch10 — Theory สำคัญ

Protection System Components

  • CT (Current Transformer) — แปลงกระแสสูง → 5A/1A
  • VT/PT (Voltage Transformer) — แปลงแรงดันสูง → 110V/115V
  • Relay — ตรวจสอบค่าผิดปกติ → ส่ง trip signal
    • CB (Circuit Breaker) — ตัดวงจร
  • Battery/DC Supply — จ่ายไฟให้ relay ทำงานได้แม้ AC fault

IDMT (Inverse Definite Minimum Time)

ยิ่ง fault กระแสสูง → ตัดวงจรเร็วขึ้น (Inverse characteristic)

Plug Setting (PS) = I_p / I_relay_rated × 100% PSM = I_fault_secondary / I_p_set = (I_fault/CTR) / I_p

Coordination / Grading (เผื่อรู้)

Coordination Time Interval (CTI) = 0.3–0.5 s Relay ใกล้ fault ทำงานก่อน (fastest) → relay บน upstream ทำงานช้ากว่า T_upstream = T_downstream + CTI
